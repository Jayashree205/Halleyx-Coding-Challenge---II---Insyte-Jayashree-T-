# Insyte – Custom Dashboard Builder

## 📌 Overview

Insyte is a **custom dashboard builder application** that allows users to create, manage, and visualize customer data dynamically. Users can build personalized dashboards, add customers, and view insights through graphical representations.

---

## 🚀 Features

* 👤 **User Authentication**

  * Secure signup and login functionality
  * Personalized dashboard for each user

* 📊 **Custom Dashboard Creation**

  * Create multiple dashboards
  * Add and manage customer data

* 📈 **Data Visualization**

  * View customer data in graphical format
  * Dynamic updates based on selected data

* 💾 **Persistent Data**

  * Saves user progress
  * Existing users can continue from where they left off

* 📱 **Responsive Design**

  * Works seamlessly on both mobile and desktop devices

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash id="0x41qj"
git clone https://github.com/Jayashree205/Halleyx-Coding-Challenge---II---Insyte-Jayashree-T-.git
cd Halleyx-Coding-Challenge---II---Insyte-Jayashree-T-
```

### 2. Install Dependencies

#### Client

```bash id="zv5jz8"
cd client
npm install
```

#### Server

```bash id="yqv5me"
cd server
npm install
```

---

### 3. Environment Variables

Create a `.env` file inside the server folder and add:

```env id="c7n2e0"
MONGO_URI=your_mongodb_connection_string
PORT=5000
```

---

### 4. Run the Application

#### Start Backend

```bash id="0ik6be"
cd server
npm start
```

#### Start Frontend

```bash id="w8w63d"
cd client
npm start
```

## 🌐 Usage

* Register as a new user → Dashboard starts empty
* Login as an existing user → Previous dashboards and data will be loaded
* Create dashboards and add customers
* Visualize data using graphs

---

## 📌 Project Behavior

* New users get an **empty dashboard**
* Existing users can **resume their work**
* Data is securely stored and linked to each user
